Bienvenido Primeros pasos en el tutorial

Antes de comenzar, vamos a comprender un poco más las virtudes y limitaciones del presente tutorial. La idea del mismo es el de brindarte una cierta noción básica sobre el Método de los Elementos Finitos, como así también aprender a utilizar correctamente FEMRIS.

Estilos de Texto

El texto de este tutorial presentará diversos formatos, cada uno de ellos indicando una interacción diferente con el usuario:

¿Ayuda?
Esta clase de texto indica información puntual sobre el objeto resaltado.
Hace referencia a que, cuando le haga click con el puntero, abrirá una descripción más detallada sobre el elemento señalizado, además de incluir referencias a fuentes externas en el Internet.
Esto indica un popover, es decir, un cuadro de texto que aparecerá encima del mismo texto cuando el usuario coloque su puntero encima del texto señalizado.
