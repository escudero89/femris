Para comenzar el proceso de utilización del Método de Elementos Finitos (FEM, Finite Element Method), debemos establecer un modelo físico con el cuál trabajar.

{{end|model}} {{start|domain}}

El segundo paso es decidir las características físicas y el dominio con el cual vamos a trabajar. En éste bloque se deben definir no solo las propiedades físicas del dominio, sino también las condiciones a las cuales se encuentra sometido.

{{end|domain}} {{start|shape_functions}}

Éste tercer paso cumple un rol educativo más que cómo parte estricta del flujo de trabajo. Descubre cómo las funciones de forma afectan un dominio 1D, y experimenta cómo sus diversos valores o funciones afectan la exactitud de la aproximación final.

{{end|shape_functions}} {{start|results}}

La última fase del proceso se enfoca en la visualización de los resultados. Analiza cómo las matrices elementales se ensamblan en cada elemento del dominio, y cómo cada valor afecta el resultado entregado al finalizar la aplicación del FEM.

{{end|results}}
