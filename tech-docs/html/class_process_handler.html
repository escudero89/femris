Detailed Description

This class handles the input/ouput of the interpreters.

It's in charge of the handling of the process related to the two interpreters availables: MATLAB and GNU Octave. It manages their inputs and outputs, and emits signals whenever there is a change in the status of the process.

void ProcessHandler::executeInterpreter ( QString  typeOfStudyCase)

Executes the interpreter.

Before calling callingProcess(), creates a file fileForProcessingInterpreter_tmp.m which contains all the information stored in currentMatFemFile.m and also the script of MATfem. That way, this new file is used as an argument for the interpreters to use as source.
