void StudyCaseHandler::adoptNewTypeStudyCaseIfNecessary ( )

This function is mainly used to check the internal structure of the Study Case.

It's call in the QML one the user change the type of the Study Case. If the type is of the different nature (e.g., heat transfer instead of structural), we create a new study case.
