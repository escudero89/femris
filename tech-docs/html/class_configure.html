Detailed Description

The Configure class that handles mainly the Configuration.

The Configure class works as a Singleton. Its main purpose is to read/write the custom configuration file for the user, which stores configuration parameters (such as which OS the user has, or what interpreter [matlab/octave] is the user using).

When the app starts, creates the custom configuration file.

This method also takes care of crashes, by setting the parameter "crashed" to "true". When the app finishes due to an unexpected behaviour, this setting remains true. This is taken into account the next time that the app starts.
